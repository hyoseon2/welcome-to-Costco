특히 사용자가 원하는 정보를 쉽게 찾기 어려운 구조적 문제로 인해 사용자 경험 (UX) 개선에 중점을 두었고,
간편한 네비게이션과 접근성 강화를 중심으로 작업을 진행했습니다.
이를 통해 사용자가 더욱 직관적으로 웹사이트를 탐색할 수 있도록 하였습니다.

Component Librery

  • 모바일 환경에서 손가락으로 터치하기 쉬운 하단에 사용자들이 가장 자주 사용하는 메뉴를 빠르게 접근할 수 있도록 설계하였습니다.
    각 아이콘은 간경하고 직관적으로 디자인 되어 사용자들이 쉽게 이해하고 이용할 수 있습니다.

  • 마우스를 상품에 올리면 제품의 다양한 모습을 한눈에 확인할 수 있고 상품의 오른쪽 상단에 있는 하트 아이콘을 클릭하면 '찜하기' 기능을 통해 사용자들에게 편리함을 제공할 수 있도록 설계하였습니다.
    특히, 상품명과 가격, 브랜드 정보 등이 시각적으로 잘 구분되어 있어 가독성이 좋고 사용자들이 상품의 중요한 정보를 빠르게 이해할 수 있습니다.

  • 사용자들이 각 카테고리를 빠르게 탐색할 수 있도록 탭 메뉴를 제공하여, 클릭 한 번으로 원하는 카테고리의 상품으로 전환할 수 있도록 설계하였습니다.
    이로 인해 불필요한 페이지 이동 없이 다양한 상품을 한 화면에서 확인할 수 있습니다. 특히, 디자인은 심플하면서도 시각적으로 잘 구분되어 있어 혼동 없이 상품의 정보를 빠르게 탐색할 수 있습니다.

  • 모바일 환경에서는 화면 크기가 제한적이기 때문에 메뉴를 기본적으로 숨겨두고 사용자가 필요할 때 메뉴를 확장시킬 수 있어, 공간을 효율적으로 활용할 수 있도록 설계하였습니다.
    불필요한 정보 과부하를 줄이면서도 필요한 정보를 쉽게 접근할 수 있고, 사용자들에게 깔끔한 탐색 경험을 제공합니다.
